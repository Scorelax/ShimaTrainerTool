# Reference Files

This folder contains the original HTML files from the Google Apps Script implementation.

These files are kept for **comparison purposes only** when updating the modern JavaScript modules.

## Files:
- **Continue_journey.html** - Original trainer selection page
- **trainer_card.html** - Original trainer card page
- **my_pokemon.html** - Original Pokemon list page
- **pokemon_card.html** - Original Pokemon details page
- **edit_trainer.html** - Original trainer edit page
- **edit_pokemon.html** - Original Pokemon edit page
- **new_journey.html** - Original new trainer form
- **new_pokemon.html** - Original Pokemon registration page
- **pokemon_form.html** - Original Pokemon form page
- **evolution.html** - Original evolution page
- **trainer_info.html** - Original trainer info page
- **conduit_*.html** - Original Conduit class pages (not yet implemented)
- **master*.html** - Original template files
- **gode.gs.js** - Original Apps Script code (deprecated)
- **examples/** - Reference examples

## Active Application
The active application files are in the **root directory**:
- `/js/pages/` - Modern modular page implementations
- `/js/api.js` - API client
- `/js/main.js` - Router and app initialization
- `/css/styles.css` - Global styles
- `/assets/` - Images and sounds

## Backend
- `Current_Code.gs` - Google Apps Script backend (deploy to script.google.com)
- `Original_Code.gs` - Original Apps Script code (reference only)
